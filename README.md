# Smart Logistics Dashboard (React + Vite)

A React + Vite dashboard built from the v25 smart logistics pipeline outputs.

## Included features

- Real project data for 100 / 200 / 1000 orders
- React state-based scenario switching
- ECharts routing comparison
- Framer Motion animated numbers
- Controlled neon emphasis for the best scenario
- Route growth animation and light particle flow in the route view
- Low-profile Logic View overlay
- Keyboard shortcuts: `1`, `2`, `3`, `D`, `Esc`
- Route illustration artifact copied into `public/assets`

## Run locally

```bash
npm install
npm run dev
```

## Notes

- This project is intentionally designed as a compact decision-support dashboard rather than a fake city control room.
- The route animation is illustrative, not a street-level navigation simulation.
- Keep the live display short if you use it during presentation.
